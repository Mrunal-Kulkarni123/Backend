# Task Management System - Backend

## Features

- User Registration
- User Login
- JWT Authentication
- Role Based Access Control
- Task CRUD Operations
- Admin APIs

## Tech Stack

- Node.js
- Express.js
- MongoDB
- JWT
- bcryptjs

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

```env
PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret
```
